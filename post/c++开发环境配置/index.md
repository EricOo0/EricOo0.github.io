# C++开发环境搭建

* 代码编辑神器 Vscode  

  1、安装拓展  

  mac自带已经有c的编译器了所以不用另外安装，终端g++/gcc --version可以查看相关信息，如果没有可以顺便安装xcode，省去一些麻烦，也可以用来写c++代码。win需要下载编译器mingw。

  Vscode只需要把c/c++相关的一些工具拓展安装下来： 

  * C/C++  

  * C/C++ Clang Command Adapter  

  * C/C++ Extension Pack 

  * C++ Intellisense  

  * C/C++ Runner  

  2、新建一个.cpp文件  
  “command+shift+p”打开命令行工具窗口，输入或者选择“Edit Configurations”命令，生成c_cpp_properties.json文件  

  也可以直接复制下面四个个文件到.vscode文件夹下  

  c_cpp_properties.json:includepath路径下可以根据自己需要配置  

```
{
  "configurations": [
    {
      "name": "macos-gcc-x64",
      "includePath": [
        "${workspaceFolder}/**"
      ],
      "compilerPath": "/usr/bin/clang",
      "cStandard": "${default}",
      "cppStandard": "c++11",
      "intelliSenseMode": "macos-gcc-x64",
      "compilerArgs": [
        "-Wall",
        "-Wextra",
        "-Wpedantic",
        "-stdlib=libc++",
        "-std=c++11"
      ]
    }
  ],
  "version": 4
}
```

* 终端  

  c++开发少不了和终端打交道，可参考mac终端美化文章进行配置

* xcode  

  苹果官方也提供了一个集成开发环境xcode，appstore可以安装--免费  

  打开xcode，选择command line tool，选择cpp语言然后就可以在项目里编译和运行c++代码了
