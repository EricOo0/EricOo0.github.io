如何使用hugo+githubpage搭建个人博客

January 14, 2022 in 指南, blog

Hugo+gitpage搭建属于自己的个人博客

对于想要

三个静态博客生成器：

  • Jekyll
  • Hugo
  • Hexo

gitpage + Hugo

1、 安装HUGO

安装指南：Install hugo

macos安装方法：

brew install hugo

Hugo version

2、建立自己的网站

进入存放网站的目录

Cd /

hugo new site my-blog

选择一个自己喜欢的模版：

Hugo Themes: https://themes.gohugo.io

将主题clone到本地

cd /my-blog # 进入网站目录 mkdir -p themes # 创建 “themes” 目录。 cd themes

git clone https://github.com/Track3/hermit.git hermit # 将 hermit 主题克隆到 hermit目录

在 Hugo 网站文件夹的根目录下/my-blog ，打开 config.toml 文件

根据所选主题编辑config.toml

新建一篇文章

hugo new post/my-first-post.md # “my-first-post.md” 是新建文章的文件名。

本地预览

hugo server -D
使用浏览器打开 http://localhost:1313 预览
如果你对预览效果满意，进入下一步。
如果不满意，编辑 config.toml 配置文件，再次预览

构建hugo网站

hugo # 构建你的 Hugo 网站，默认将静态站点保存到 “public” 目录。

转化为git库

cd ~/my-blog/public # 生成的 HTML 文件保存在 “public” 目录中，“public” 文件夹会被转换为 Git 库。 git init # 初始化 Git 库。

关联到远方库

git remote add origin https://github.com/EricOo0/EricOo0.github.io.git

git remote add origin git@github.com:EricOo0/EricOo0.github.io.git

git status # 查看当前修改状态。 git add . # 添加所有修改过的文件。你也可以只添加某个文件。 git commit -m “Add a new post” # “Add a new post” 是 commit message.

git push -u origin master
